There are a few different options to build Jekyll. The simplest way to is simply open a Terminal/Command window inside your Jekyll folder and type gulp.

You can also run gulp jekyll-build. This will build the Jekyll site expose the UI editor. This editor is available at http://localhost:4000/admin.

Jekyll Admin UI Editor

The admin interface allows people to edit portions of the site without working directly with code. For example, a user cannot directly edit a page.html file but they can edit any translations, data file or collections that are used on the page.

For example, on the All Solutions page while a non-technincal user would not be able to edit the layout of the page, they would be able to edit:

  • All text and translations
  • Product cards stored in the _products collections
  • Customer testimonials stored in data files
If Optimizely is available on the page, a non-technical user would also be able to change the position of each block of content on the page.
